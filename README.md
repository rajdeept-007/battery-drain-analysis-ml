# Smartphone Battery Drain Prediction Using Machine Learning

## Overview
This project analyzes smartphone usage patterns and predicts battery drain using Machine Learning techniques.

The project includes:
- Exploratory Data Analysis (EDA)
- Simple Linear Regression
- Multiple Linear Regression
- Logistic Regression

## Features Used
- Screen Brightness
- Gaming Hours
- App Usage Time
- GPS Usage
- Bluetooth Usage
- WiFi Usage
- Video Streaming Hours
- Charging Frequency

## Objectives
- Predict battery drain percentage
- Analyze factors affecting battery consumption
- Classify fast battery drain conditions

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
